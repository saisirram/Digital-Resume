Real Time Blogging Application

  • Source code
Technologies: Java, Spring Boot, Spring Data JPA, Spring Security, JWT, MySQL, Swagger, and AWS.

Project Description:

Java based Restful web services developed to allow users to post their content based on roles and authorities, read and comment on other users blogs.

Responsibilities:

  • Designed and developed REST APIs for users to Post their content, read and comment on blog posts.
  • Secured the application, restricted the access of users based on role and JWT token using spring security.
  • Deployed the JAR file in AWS using Elastic Beanstalk Service.
  • Tested the REST APIs using Swagger UI.
